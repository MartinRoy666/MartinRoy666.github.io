Réalisations

Api rest

Développement d'un api pour location de voiture. Le système permet à des clients de louer des voitures pour une période choisie. Il peut consulter le catalogue des voitures et lorsqu'une location est effectuée, une facture est générée avec les informations du client. Des recherches sur les voitures sont disponible et qui ont été développé avec Graph Ql. L'api est écrit en javascript avec Node js et avec la librairie Express Js. La base de donnée est NoSql.

React
